### 导入

> * 手动导入：
> 
>> * **SwpModel** 文件夹导入项目中。
>> * **`#import "SwpModelHeader.h`**
>> 
> -------

> * CocoaPods 导入:
> 
>> * **pod search SwpModel**
>> * **pod 'SwpModel'**
>> * **`#import <SwpModel/SwpModelHeader.h>`**
>> 
> -------
